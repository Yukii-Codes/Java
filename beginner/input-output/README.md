# Input/Output Operations

This folder contains Java programs that demonstrate how data is taken as input and produced as output using console and file-based operations.

## Topics Covered
- [ ] Console input and output
- [x] Using Scanner for input
- [ ] Using BufferedReader for input
- [ ] System.out and System.err
- [ ] File handling basics
- [ ] FileReader and FileWriter
- [ ] BufferedReader and BufferedWriter
- [ ] Input and output streams (basic concepts)

## Goal
To understand how data flows between a Java program and external sources such as the keyboard, console, and files.
