# Control Statements

This folder contains Java programs that demonstrate how to control the flow of program execution using conditional and looping constructs.

## Topics Covered
- [x] Decision-making statements (if, if-else, else-if, switch)
- [x] Looping statements (for, while, do-while)
- [ ] Enhanced for loop (for-each)
- [x] Branching statements (break, continue, return)
- [x] Nested control statements
- [x] Loop control concepts

## Goal
To understand and effectively use control statements to make decisions, repeat tasks, and manage program flow in Java.
