# Exception Handling

This folder contains Java programs that demonstrate how to handle runtime errors and exceptional conditions in a program.

## Topics Covered
- [ ] What is an exception
- [ ] Types of exceptions (checked and unchecked)
- [ ] Exception hierarchy
- [ ] try and catch blocks
- [ ] Multiple catch blocks
- [ ] finally block
- [ ] throw keyword
- [ ] throws keyword
- [ ] User-defined (custom) exceptions

## Goal
To understand how exceptions work in Java and to handle errors gracefully without crashing the program.
