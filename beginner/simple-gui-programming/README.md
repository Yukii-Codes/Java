# Simple GUI Programming

This folder contains Java programs that demonstrate the basics of building graphical user interfaces (GUI) using AWT and Swing.

## Topics Covered
- [ ] Introduction to GUI and event-driven programming
- [ ] AWT components (Frame, Label, Button, TextField)
- [ ] Swing components (JFrame, JLabel, JButton, JTextField, JTextArea)
- [ ] Layout managers (FlowLayout, BorderLayout, GridLayout)
- [ ] Event handling using ActionListener
- [ ] Dialog boxes using JOptionPane
- [ ] Simple GUI applications (basic forms and interactions)

## Goal
To understand how Java applications interact with users through graphical interfaces and to build simple, functional GUI-based programs.
