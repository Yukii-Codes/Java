# Object-Oriented Programming (OOP)

This folder contains Java programs that demonstrate object-oriented programming concepts and how they are implemented in Java.

## Topics Covered
- [x] Classes and Objects
- [x] Methods (declaration and calling)
- [x] Encapsulation
- [x] Inheritance
- [x] Polymorphism
- [x] Abstraction
- [x] Constructors (default and parameterized)
- [x] Method overloading and method overriding
- [x] Keywords: this, super, static, final
- [x] Abstract classes and interfaces
- [ ] Packages and access modifiers
- [ ] Inner classes (basic)

## Goal
To understand and apply object-oriented principles in Java to build modular, reusable, and maintainable programs.
